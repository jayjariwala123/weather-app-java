# 🌦 Weather App (Java)

A command-line weather application using Java and OpenWeather API.

## Features
- Fetch real-time weather data
- Displays temperature, humidity, wind speed
- Handles invalid input
- Loop-based interaction

## Tech Stack
- Java
- HTTP Client API
- Gson
- OpenWeather API

## Run

javac -cp .:lib/gson-2.10.1.jar WeatherApp.java  
java -cp .:lib/gson-2.10.1.jar WeatherApp  

## Note
Set your API key using environment variable:
WEATHER_API_KEY
